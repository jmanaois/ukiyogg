JobHunter

August 2022 - December 2022
Software Development Web App SQL JavaScript

JobHunter is a web application that I helped design with 4 other Software Developers for my Software Development class. For this project, we practiced going through the entire Software Development life cycle and aimed to develop a website that allows students to search for and apply for jobs in the technology industry. For this project, I managed the application's backend database to make sure data was being safely entered and recalled. I also maintained the GitHub repository and was responsible for the project's documentation.

The application's main feature is its search and filter function. Utilizing SQL for the backend of the application, all the data is stored and recalled from their respective tables. For instance, users are separated into Companies and Students, and their login information is stored in the userData table. Job postings are stored in their own table as well, and the search function calls that information when a user is searching for either specific keywords or filter.

We intended for the application to allow users to customize their profiles and set alerts for job postings, however due to time constraints we were unable to get the feature tested and deployed in time.

The website also allows Company users to post job listings. Once posted, the data is stored into the Posting table and the listing can be searched in the home page's search bar.

At the moment, the site is not being hosted, but you can check out the GitHub repository here.
